# App项目简介

``本App主要实现了新闻资讯、短视频、发布视频、聊天、个人主页等功能，如下：``

## 1、新闻资讯

- 获取**新闻资讯**，包括“国际”，“科技”，“教育”，“娱乐”等8个分类
- 获取**微博热搜榜单**
- 获取全国以及各地区**疫情实时数据**

## 2、短视频

- 获取**小屏视频流**
- 获取**全屏视频流**

## 3、发布视频

- 拍摄**照片**
- 拍摄**视频**
- **上传视频及封面**

## 4、聊天消息

- 获取**联系人列表**
- 实现**聊天信息**
- 随机模拟**发送**，**收取**消息

## 5、个人主页

- 登录**账号**
- 展示**个人信息**
- 修改**个人资料**
- **退出登录**
- **工具列表**
